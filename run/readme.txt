W katalogu znajduje katalog projektu środowska testowego testingEnvironment. Zawiera on katalogi src i target oraz plik pom.xml zgodnie z strukturą projektów w jezyku Java. Katalog target zawiera m.in. plik testingEnvironment-1.0.jar który należy uruchomic w celu testu środowiska. W lokalizacji /testingEnvironment/src/main/java/com/mycompany/testingenvironment znajduje sie kod żródłowy śroowiska testowego:
-klasa App.java -klasa uruchomieniowa
-klasa Rows -defiunująca opis jednego znalezionego znaku 
-klasa Point -reprezentuje jeden pixel dla danego obrazu
-klasa Test -implementująca calą logike środowiska testowego
